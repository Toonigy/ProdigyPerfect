Frequently Asked Questions

Why can't I do Rune Runs in Harmony Island in 3-15-3?

Rune Runs require a specific API connection that is no longer accessible for this version of the game. Unfortunately, we can't get the API necessary to enable that feature at this time.

Is my data saved?

Currently, these versions are designed for local and individual play. Since we are using modified legacy builds, integration with official cloud saves is not supported.

Authentic Experience

We don't just host old files; we ensure they function properly on modern hardware while maintaining original assets.

Enhanced Features
